# Concrete Volume Calculator

A free, fully client-side concrete estimator. Enter slab/footing or round column/tube dimensions and instantly get the volume in **cubic yards, cubic metres, cubic feet** and the **number of pre-mixed bags** needed — with an optional waste allowance. Works in imperial or metric.

**Live site:** https://harveyai26032026-ai.github.io/concrete-volume-calculator/

## Features
- Rectangular slab/footing and round column/tube modes
- Imperial (ft/in) and metric (m/cm) units
- Quantity multiplier + 0/5/10% waste allowance
- Bag count for 80 lb / 60 lb / 40 lb bags
- 100% client-side, mobile-first, no dependencies

## Files
- `index.html` — markup, SEO meta, Open Graph, FAQ + WebApplication schema
- `style.css` — clean white/light responsive theme
- `app.js` — volume + bag math (verified against known values)
- `sitemap.xml`, `robots.txt`
